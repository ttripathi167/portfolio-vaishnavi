Home Credit Default Risk

Project Overview

The Home Credit Default Risk Project aims to predict loan default risk, helping lenders make informed decisions. This project leverages multiple datasets to uncover patterns in customer behavior and loan repayment, focusing on minimizing financial risks for lenders.

Problem Statement

The main challenge is predicting whether a loan applicant will default. This binary classification task faces an imbalance issue, with the majority of loans repaid and a smaller proportion defaulting. Accurate prediction is crucial for lending decisions to minimize losses.

Solution/Approach

  • Data Cleaning: Handled missing values and dropped columns with high missingness (over 50%). Applied imputation techniques for moderate missing data.
  • Feature Engineering: Created new features that add predictive value, including:
    • Debt-to-credit ratio from bureau.csv
    • Maximum utilization rate for credit cards from credit_card_balance.csv
    • Total number of active credits
    • Days since the last loan application
  • Model Selection: Evaluated models like Logistic Regression, Random Forest, and Gradient Boosting (GBM), balancing interpretability, complexity, and predictive power.

Key Steps and Tasks Performed

  • Exploratory Data Analysis (EDA): Conducted target distribution analysis, correlation analysis, and examined class imbalance.
  • Data Preprocessing: Managed missing values, applied transformations for skewed data, and reduced dimensionality using feature selection.
  • Modeling and Evaluation: Trained various models, undersampled the majority class, and optimized hyperparameters for improved recall on default predictions.

Tools & Skills Utilized

  • Programming Languages & Libraries: Python (Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn)
  • Machine Learning Models: Logistic Regression, Random Forest, Gradient Boosting
  • Data Processing: Feature engineering, handling imbalanced data, and outlier management

Key Learnings

This project enhanced skills in handling imbalanced datasets, feature engineering across diverse data sources, and model tuning for classification accuracy. Gained experience with advanced data cleaning techniques and feature transformation to maximize model performance.
