Predicting Stock Prices Using Scalable AWS Infrastructure

Project Overview

This project leverages the AWS ecosystem to build a scalable infrastructure for predicting stock prices. Using Alpha Vantage for stock data, AWS Lambda, Glue, and S3 for data management, and SageMaker for model training, this setup provides real-time analytics to assist financial institutions in decision-making.

Problem Statement

Financial data processing and analysis can be challenging due to the high volume and real-time requirements. Traditional infrastructure often fails to meet these demands, resulting in delayed insights and missed opportunities. This project addresses these challenges by creating a scalable solution for real-time stock data analytics.

Solution/Approach

  • Data Collection: Created Lambda function using AWS Lambda that fetches data from Alpha Vantage API, stores in S3, and is scheduled through EventBridge for daily updates.
  • Data Cleaning: Utilised AWS Glue to create ETL jobs which cleans and transforms the data to create it in structured format for further processing
  • Modeling: Used Amazon SageMaker Notebook instance for development and testing the code, further created training jobs to run the python scripts where I used Lasso Regression models on cleaned data to make stock prediction for Magnificent 7 stocks. The ouput is updated to S3.
  • Monitoring and Logging: Used Cloudwatch to monitor the jobs and checking logs while development to test if the process is running as expected.
  • Scheduling: Scheduled the entire pipeline to run daily after market close through Amazon EventBridge.

Key Steps and Tasks Performed

  • Exploratory Data Analysis (EDA): Analyzed target distributions, correlations, and performed data balancing.
  • Data Preprocessing: Managed missing values, applied transformations, and performed outlier analysis.
  • Modeling and Evaluation: Trained models, optimized hyperparameters, and evaluated results.

Tools & Skills Utilized

  • Python (Pandas, NumPy, Scikit-Learn)
  • AWS Services: Lambda, S3, Glue, EventBridge, SageMaker
  • Machine Learning with Lasso Regression

Key Learnings

Gained proficiency in using AWS services for scalable data processing and real-time analytics. Enhanced skills in machine learning applications for financial forecasting.
